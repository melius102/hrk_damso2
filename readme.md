## WebRTC 및 언어번역 챗팅 사이트 개발

URL: https://damso.herokuapp.com/

### 적용 기술
1. Front-End
    - Web API (WebRTC, Media Stream)
    - jQuery
    - Handlebars
    - Socket.IO (Client)

2. Back-End
    - Express (Node.js)
    - Socket.IO (Server)
    - Mongoose
    - NAVER & Kakao API (Translation)
    - Heroku (Hosting)
